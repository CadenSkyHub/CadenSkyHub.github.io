# Navigator/History



## Navigator

主要用于获取浏览器信息

`userAgent`

`language`



### History

主要管理历史记录，该对象与浏览器的地址栏操作相互对应，如后退，前进，历史记录等

| 方法            | 说明                                                         |
| --------------- | ------------------------------------------------------------ |
| back()          | 后退                                                         |
| forword()       | 前进                                                         |
| go(参数:number) | 前进后退功能，如果为 `1` 则前进 `1` 个页面<br />如果为 `-1` 则后退 `1` 个页面 |
